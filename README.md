# aiml-project

An ML-powered tool that predicts AQI risk levels and warns about lung diseases caused by air pollution.

## How it works:
Enter pollution readings (PM2.5, PM10, CO, etc.) and the tool predicts the AQI category and shows health risks, lung impact and safety advice.

You can get real readings from https://aqicn.org

## Dataset:
Air Quality Data in India — Kaggle (26 cities, 2015–2020)
